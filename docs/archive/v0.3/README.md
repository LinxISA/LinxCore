# Linx ISA v0.3 Uop Snapshot (Archived)

This directory preserves the retired v0.3 LinxCore uop-classification snapshot.
It is non-normative and intentionally contains historical instruction names.

The active v0.56.5 classification is owned by the LinxISA superproject at
`isa/v0.56/uop_classification_v0.56/`. Do not copy identities or encodings from
this archive into current decoders, generators, or documentation.
